Frases e Áudios

  • Saudação

    Hello / Hi. (Olá)

    Use para cumprimentar alguém de forma casual ou formal.

  • Saudação

    Good morning. (Bom dia)

    Ideal para cumprimentar alguém pela manhã.

  • Saudação

    How are you? (Como vai você?)

    Pergunta comum para iniciar uma conversa.

  • Cortesia

    Thank you. (Obrigado)

    Use para expressar gratidão em qualquer situação.

  • Cortesia

    Excuse me. (Com licença)

    Perfeito para chamar atenção educadamente ou pedir passagem.

  • Viagem

    Where is the bathroom? (Onde fica o banheiro?)

    Frase essencial para emergências em locais públicos.

  • Viagem

    I'm lost. (Estou perdido.)

    Use para pedir ajuda ao se perder em uma cidade ou local desconhecido.

  • Emergência

    Call the police! (Chame a polícia!)

    Frase importante para situações de emergência.

  • Cotidiano

    Can you help me, please? (Você pode me ajudar, por favor?)

    Útil para pedir assistência em qualquer situação.

  • Viagem

    How much is it? (Quanto custa?)

    Pergunta essencial para compras e negociações.

  • Viagem

    I have a reservation. (Eu tenho uma reserva.)

    Use ao fazer check-in em hotéis ou restaurantes.

  • Cortesia

    I'm sorry, I don't understand. (Desculpe, eu não entendo.)

    Frase útil para pedir esclarecimentos em conversas.

  • Cotidiano

    I have no idea. (Eu não faço ideia.)

    Use para expressar que você não sabe algo.

  • Cotidiano

    I'll be right there! (Estarei aí!)

    Frase para avisar que você está a caminho.

  • Cotidiano

    I'll get it. (Eu pego.)

    Use para se oferecer para pegar algo ou atender.

  • Cotidiano

    I'm on my way out. (Estou saindo.)

    Frase para informar que você está de saída.

  • Cotidiano

    I'm running late. (Estou atrasado.)

    Use para avisar que está atrasado para um compromisso.

  • Cotidiano

    It's been a long day and we're completely whacked. (Foi um dia longo e estamos completamente exaustos.)

    Frase para expressar cansaço após um dia difícil.

  • Cotidiano

    It's up to you. (Você decide.)

    Use para deixar a decisão nas mãos de outra pessoa.

  • Cortesia

    Let me know if you have changed your plans. (Me avise se você mudou seus planos.)

    Frase para confirmar mudanças de planos.

  • Cotidiano

    Let's call it a day. (Vamos encerrar por hoje.)

    Use para sugerir encerrar atividades do dia.

  • Cortesia

    No worries. (Sem problemas.)

    Frase para tranquilizar alguém ou minimizar um problema.

  • Cotidiano

    Sounds like a plan. (Parece um bom plano.)

    Use para concordar com uma sugestão ou ideia.

  • Cortesia

    Take care. (Cuide-se.)

    Frase comum para se despedir de alguém com carinho.

  • Saudação

    Yo, what's going on? (E aí, o que está acontecendo?)

    Saudação casual para amigos ou conhecidos.
